# BronchoTransformer
Para poder probar el modelo, se debe descargar el dataset sintético del paper BronchoPose de Borrego et.al el cual lo puede conseguir aquí:
https://dataverse.csuc.cat/dataset.xhtml?persistentId=doi:10.34810/data2251
Una vez descargado se debe unir los split zips para poder tener el dataset completo. 
En el caso de que use otro tipo de dataset ya sea sintético o real, se deben hacer los respectivos cambios a los códigos.
